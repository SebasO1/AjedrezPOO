# Proyecto de Ajedrez en Java

Proyecto realizado en Java con tecnología Maven como proyecto final de la asignatura **POO**.  
Es un ajedrez básico que contiene las mecánicas de mover, comer y cambiar de turno.

## Autores
- Emmanuel Bonilla Mitrotti
- Diego Tique
- Sebastián Olaya

## Tecnologías utilizadas
- Java (OpenJDK 25)
- Maven
- VS Code / NetBeans

## Instalación
1. Clonar este repositorio:
   ```bash
   git clone https://github.com/usuario/proyecto-ajedrez.git
2. Entrar en la carpeta del proyecto:
    cd proyecto-ajedrez
3. Compilar con Maven:
    mvn clean install
## Uso
Ejecutar el proyecto con:
    mvn exec:java

## Funcionalidades
- Movimiento básico de piezas.
- Captura de piezas.
- Cambio de turno entre jugadores.
